# 네이버 상위글 3시간 강화 루프

## 목적

네이버 VIEW 상위 노출 글이 계속 바뀌므로, 고정 프롬프트를 장기 사용하지 않는다.
3시간 간격으로 검색 결과를 다시 확인하고, 새 패턴을 기존 프롬프트에 반영한다.

## 운영 규칙

- 기본 주기: 3시간
- 각 실행마다 새 폴더를 만든다
- 최근 3회 분석 결과를 함께 비교한다
- 반복되는 상투 패턴은 약화시키고, 새로 떠오른 검색 의도는 강화한다

## 매 회차 산출물

- `search_notes.md`
- `analysis.md`
- `claude_prompt.md`
- `draft.txt`
- `review.md`

## 강화 포인트

- 새로 노출되는 제목 리듬
- 가격 범위 표현 변화
- `위고비 마운자로 가격`, `위고비 알약 가격`, `위고비 가격 인하` 같은 연관 수요 변화
- 경험담 비중이 커졌는지, 정보형 비중이 커졌는지
- 가격 구조를 더 깊게 물어보는 흐름인지, 비교 수요가 커졌는지

## 프롬프트 갱신 방식

- 유지할 규칙과 버릴 규칙을 분리한다
- 최근 3회에서 2회 이상 반복된 패턴만 핵심 규칙으로 승격한다
- 한 번만 보인 자극형 문구는 노이즈로 취급한다
- 상위글이 광고성 대체재로 빠질수록, 우리 프롬프트는 가격 정보 밀도를 더 높인다

## 초안 루프

1. Codex가 cmux 로 상위글 수집
2. Codex가 분석본과 강화 프롬프트 작성
3. Claude가 원고 작성
4. Codex가 원고 리뷰와 수정 지시 작성
5. 필요하면 Claude에게 2차 수정 요청
