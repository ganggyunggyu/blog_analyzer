# 위고비 가격 네이버 상위글 1차 분석

작성 시각: 2026-04-09 15:00 KST
수집 방식: cmux browser 기반 네이버 VIEW 검색 + 모바일 블로그 본문 확인
메인 키워드: 위고비 가격

## 검색 결과 레이어 관찰

- VIEW 검색인데도 카페, 블로그, 웹 FAQ, 지식인, 위키류가 섞여 있음
- 연관 키워드로 `위고비 마운자로 가격`, `위고비 알약 가격`, `위고비 처방 가격`, `위고비 한달 가격`, `위고비 가격 인하` 가 강하게 붙음
- 커뮤니티 카드가 상단에 끼어들고, 실제 블로그 글은 `경험담 + 부작용 + 가격` 조합이 많이 노출됨

## 확인한 네이버 블로그 글

1. 위고비 가격 처방 다이어트주사 후기 12kg 감량
2. 위고비 가격 다이어트약 부작용 (50kg)
3. 위고비 가격 처방 식욕억제제 부작용 (꿀팁)

## 공통 구조

- 제목 첫머리에 메인 키워드 `위고비 가격` 을 그대로 둠
- 제목 뒤에 `처방`, `부작용`, `후기`, `감량`, `식욕억제제` 같은 검색 의도어를 붙임
- 도입부는 거의 모두 실제 체중 증가와 감량 서사로 시작함
- 초반 20~30퍼센트 구간 안에 처방 이유, 가격, BMI 조건을 빠르게 배치함
- 중간에는 효과와 부작용을 나열식으로 설명함
- 후반부는 위고비 자체보다 보조제나 대체 루틴으로 중심이 이동하는 경우가 많음

## 제목 패턴

- 실사용 후기형이 압도적임
- 숫자 또는 체중 변화 수치를 붙여 클릭을 유도함
- 가격만 다루는 정보글처럼 보이지만 실제로는 감량 후기형 제목이 많음
- 비교형보다는 `경험형 + 경고형` 이 더 많음

## 도입부 패턴

- 살이 얼마나 쪘는지, 언제부터 힘들었는지, 몇 kg를 뺐는지를 먼저 밝힘
- 극단적인 감량 동기가 먼저 나오고 그 뒤에 위고비가 등장함
- 가격 키워드인데도 가격표보다 감정선이 먼저 들어감

## 가격 정보 배치

- `1펜`, `처방비 별도`, `30만~40만원대`, `용량이 높아질수록 상승` 표현이 반복됨
- 병원 처방비와 약국 구매비를 분리해서 적는 패턴이 있음
- 장기 유지 시 비용 부담이 커진다는 문장을 거의 반드시 붙임
- 가격을 단순 수치보다 `계속 맞기엔 부담스럽다` 는 판단형 문장으로 처리함

## 처방/조건 패턴

- BMI 27 이상 + 동반 질환 또는 BMI 30 이상 문장을 반복적으로 사용함
- `전문의 상담`, `비급여`, `조건이 까다롭다` 같은 안전장치 문구가 들어감
- 최신 허가 범위보다 블로그식 요약 문장이 많아서 표현이 서로 비슷함

## 부작용 패턴

- 메스꺼움, 울렁거림, 구토, 설사, 속쓰림, 불면증이 반복됨
- `췌장염`, `요요`, `중단 후 식욕 복귀` 같은 무거운 리스크를 넣음
- 효과보다 부작용 체감이 더 셌다는 서술이 자주 보임

## 글의 약점

- 가격 키워드인데 실제 가격 정보 밀도가 높지 않음
- 광고성 대체 보조제 소개로 중후반이 흐려짐
- 가격, 처방, 유지비, 부작용, 중단 후 관리가 서로 섞여 있음
- 독자가 가장 궁금한 `그래서 얼마가 들고 누구에게 맞는가` 가 뒤로 밀림

## 우리가 가져갈 것

- 제목 첫머리 메인 키워드 정확 일치
- 초반부에 가격과 처방 조건을 빠르게 배치하는 리듬
- 병원비, 약값, 유지비를 분리해서 설명하는 방식
- 중단 후 요요와 유지 계획까지 같이 언급하는 현실적인 관점

## 우리가 피할 것

- 대체 보조제 판매 흐름으로 빠지는 구조
- 지나치게 긴 체중 변화 서사
- 가격 키워드인데도 정보보다 감정과 후기만 과한 구성
- 효과/부작용 리스트만 기계적으로 붙이는 문장

## 프롬프트 강화 방향

- 메인 의도를 `가격 확인형 + 처방 현실형` 으로 고정
- 약값, 초진/재진, 검사비, 증량 후 유지비를 분리해서 적게 만들 것
- `싸다/비싸다` 단정 대신 비용 구조와 병원별 차이를 설명하게 만들 것
- 위고비와 마운자로 비교 수요를 보조 문단으로 흡수할 것
- 후기형 톤은 유지하되 보조제 유도나 광고성 전환은 금지할 것
- 글 후반은 `누가 시작 전에 무엇을 확인해야 하는지` 로 닫을 것
